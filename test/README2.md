# Monty-Hall-Problem. Ein interaktive Python-Simulation *Eine in Python erstellte Befehlszeilensimulation des berühmten Monty-Hall-Problems.* *Mit diesem Projekt können Sie mehrere Runden spielen, Türen wechseln, Ihre Punktzahl speichern und beobachten, wie sich die Wahrscheinlichkeit im Laufe der Zeit entwickelt.* --- ## Was ist das Monty-Hall-Problem? Das Monty-Hall-Problem ist ein Wahrscheinlichkeitsrätsel, das auf einem Spielshow-Szenario basiert: 1. Sie wählen eine von drei Türen aus. 2. Eine Tür, die Sie nicht ausgewählt haben, wird geöffnet und hinter ihr befindet sich eine Ziege. 3. Sie können bei Ihrer ursprünglichen Wahl bleiben oder wechseln. 4. Statistisch gesehen verdoppelt ein Wechsel Ihre Gewinnchancen auf das Auto (von 33 % → 66 %). Mit diesem Projekt können Sie die Mathematik interaktiv erleben. ## Funktionen: - [x] Vollständig interaktives CLI-Gameplay - [x] Automatische Enthüllung der Ziege nach den korrekten Monty-Hall-Regeln - [x] Live-Verfolgung von Gewinnen/Verlusten - [x] Spielen Sie unbegrenzt viele Runden, bis Sie aufhören - [x] Eingabevalidierung für beide Auswahlmöglichkeiten - [x] Zusammenfassung und Gewinnwahrscheinlichkeit am Ende ## So funktioniert die Simulation **1. play_round()** Behandelt eine einzelne vollständige Runde: - Fragt nach der anfänglichen Türwahl - Wählt zufällig die Position des Autos aus - Monty enthüllt eine Ziege hinter einer zulässigen Tür - Fragt nach Ihrer endgültigen Wahl - Bestimmt Gewinn/Verlust *Gibt zurück:* True -> Gewinn False -> ungültige Eingabe oder Verlust None -> Beendigungssignal **2. main()** - Führt kontinuierliche Runden des Spiels aus: - Verfolgt Gewinne und Gesamtzahl der Spiele - Erlaubt unbegrenzte Runden - Druckt die Punktzahl nach jeder Runde aus - Gibt die endgültige Leistung beim Beenden aus ## Ausführen des Programms **Anforderungen:** Python 3.8+ Nur Standardbibliothek (keine zusätzlichen Abhängigkeiten) *Ausführen:* python montyhall.py Sie werden aufgefordert, Türen auszuwählen und Runden zu spielen. Drücken Sie Strg+C oder senden Sie EOF (Strg+D unter Linux/macOS, Strg+Z unter Windows), um das Programm zu beenden. --- ## Autor Bebe Egerton bebeegerton14@gmail.com
