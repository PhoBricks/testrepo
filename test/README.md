# Monty-Hall-Problem – Eine interaktive Python-Simulation

*Eine in Python erstellte Befehlszeilensimulation des berühmten Monty-Hall-Problems.*

Mit diesem Projekt können Sie mehrere Runden spielen, Türen wechseln, Ihre Punktzahl speichern und beobachten, wie sich die Wahrscheinlichkeit im Laufe der Zeit entwickelt.

---

## Was ist das Monty-Hall-Problem?

Das Monty-Hall-Problem ist ein Wahrscheinlichkeitsrätsel, das auf einem Spielshow-Szenario basiert:

1. Sie wählen eine von drei Türen aus.
2. Eine Tür, die Sie nicht ausgewählt haben, wird geöffnet und hinter ihr befindet sich eine Ziege.
3. Sie können bei Ihrer ursprünglichen Wahl bleiben oder wechseln.
4. Statistisch gesehen verdoppelt ein Wechsel Ihre Gewinnchancen auf das Auto (von 33 % auf 66 %).

Mit diesem Projekt können Sie die Mathematik interaktiv erleben.

## Funktionen

- [x] Vollständig interaktives CLI-Gameplay
- [x] Automatische Enthüllung der Ziege nach den korrekten Monty-Hall-Regeln
- [x] Live-Verfolgung von Gewinnen/Verlusten
- [x] Spielen Sie unbegrenzt viele Runden, bis Sie aufhören
- [x] Eingabevalidierung für beide Auswahlmöglichkeiten
- [x] Zusammenfassung und Gewinnwahrscheinlichkeit am Ende
